# NeoSpend

Plataforma de gestão financeira pessoal em desenvolvimento para consolidar receitas, despesas e metas em um único ambiente, com ênfase em governança de dados, previsibilidade e suporte à decisão.

## Visão do produto
- **Posicionamento**: painel financeiro completo, orientado a métricas e evolução contínua.
- **Potencial**: base sólida para projeções, alertas inteligentes, metas e integrações.
- **Foco atual**: núcleo confiável para registro, classificação e leitura de informações financeiras.

## Estado do projeto
- Estrutura Angular moderna, preparada para expansão por módulos.
- Base técnica pronta para escalar funcionalidades e cobertura de testes.

## Stack
- Angular
- TypeScript
- Vite + Vitest (testes)

## Execução local
```bash
npm install
npm start
```

Aplicação disponível em `http://localhost:4200/`.

## Scripts
```bash
# desenvolvimento
npm start

# build de produção
npm run build

# testes unitários
npm test
```
